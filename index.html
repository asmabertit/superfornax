Moderation Commands

Fornax provides a suite of moderation commands to help maintain order and enforce rules on your server. These commands ensure you can manage user behavior effectively:

Commands:

  • +warn: Warn a user for inappropriate behavior
  • +mute: Temporarily prevent a user from speaking
  • +unmute: Remove a mute from a user
  • +kick: Remove a user from the server
  • +ban: Permanently ban a user from the server
  • +vb: Verify a user as a boy
  • +vg: Verify a user as a girl
  • +role: Assign a role to a user
  • +jail: Temporarily restrict a user’s permissions
